<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args → settings.json | `config set`/`preset` write user input into the persisted config |
| settings.json → launchd plist | `config apply` regenerates an OS-level scheduled job |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-44-11 | Tampering | config set arbitrary key | mitigate | set whitelists known settable keys; unknown keys (and any core address) rejected with exit 1 |
| T-44-12 | Tampering | config set core-disabling value | mitigate | post-write loadMergedConfig re-check; D-12 guardrail strips it; CLI warns it was rejected |
| T-44-13 | Elevation | plist regeneration | mitigate | reuse the existing installMacOSScheduler bootout→enable→bootstrap; only the StartInterval integer is templated from a numeric sleepFrequencyHours (coerced), no shell interpolation of user strings |
| T-44-14 | Tampering | plist injection via frequency | mitigate | frequency coerced to integer seconds (parseInt) before substitution; non-numeric → fallback default, never raw string into the plist |
</threat_model>
