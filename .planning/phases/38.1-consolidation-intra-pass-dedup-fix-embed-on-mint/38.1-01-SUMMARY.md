## Deviations from Plan

### Deviation 1 (Rule 1 - Bug) â€” consolidation_event assertion removed from DEDUP-01 test

**Found during:** Task 3, first test run.

**Issue:** The RESEARCH proposed asserting `SELECT COUNT(*) FROM consolidation_event WHERE event_type LIKE 'contradict%'` as a belt-and-suspenders check. However, the test harness constructs `Consolidator` without a sink argument (defaults to `NoopConsolidationSink`), which does not write to `consolidation_event`. The assertion returned 0 even when tombstone correctly triggered.

**Fix:** Removed the `consolidation_event` assertion. The `tombstoned >= 1 AND judgeCalls >= 1` assertions are sufficient and more direct: tombstone proves the contradict path executed, judgeCalls proves B escalated to the judge (not auto-unrelated).

**Files modified:** `tests/consolidator.test.ts` (assertion removed before commit).
