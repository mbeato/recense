### WR-01: DEDUP-01 test relied on `judge()` override the production path does not call directly

**Files modified:** `tests/consolidator.test.ts`
**Commit:** 5dbfab4
**Applied fix:** Added an explicit `judgeBatch` override to `CandidateCapturingProvider` that delegates to `this.judge()` via `Promise.all(items.map(...))`. Verified: `npm test -- --run tests/consolidator.test.ts` stays 8/8 green; `tsc` (run via `pretest` build) passed with no type errors.
